# is-odd-rs

A program (or a library) to check if an integer is odd or even.
written in rust.

USAGE:

```bash
    is-odd [NUMBER] # where number any integer
```

EXAMPLES:

```bash
    is-odd 5 # 5 is odd
```

```rust
    is-odd-rs::is_odd(5); // true
    let int = 3;
    int.is_odd(); // true
```
